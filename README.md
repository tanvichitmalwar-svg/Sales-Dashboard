# Sales-Dashboard
1. Project Title / Headline :
Sales Analytics : Sales Analysis Dashboard
A dynamic, interactive data visualization tool built to explore sales report data worldwide-focusing on regional comparisons.

2. Short Description / Purpose :
This project focuses on building an interactive Power BI dashboard to analyze business data and generate actionable insights. The dashboard helps stakeholders understand key performance metrics such as sales trends, product performance, and overall business productivity through clear visualizations.

The goal of this project is to transform raw data into meaningful insights that support data-driven decision making.

3. Technologies Used :
*Power BI – Data visualization and dashboard creation
*Power Query – Data cleaning and transformation
*DAX (Data Analysis Expressions) – Creating calculated measures and KPIs
*Excel / Dataset – Data source for analysis

4.Key Features of the Dashboard :
*Interactive filters and slicers for dynamic analysis
*KPI cards showing important business metrics
*Sales performance visualization
*Product/category level insights
*Trend analysis using charts and graphs
*Clean and user-friendly dashboard layout

5.Insights Generated :
*Identification of top-performing products/categories
*Sales trends across different time periods
*Business performance comparison across segments
*Key indicators that influence business productivity
